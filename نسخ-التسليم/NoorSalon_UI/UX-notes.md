# NoorSalon UI/UX Direction

## الهدف
تصميم واجهة عربية RTL لإدارة صالون حلاقة، تركيزها الأساسي هو سرعة إدخال الإيرادات اليومية ومراجعة أرصدة الموظفين والموردين بدون ازدحام بصري.

## الهيكل المقترح
- شاشة دخول بسيطة تحمل هوية NoorSalon.
- شريط جانبي ثابت للتنقل بين: لوحة التحكم، الإدخالات، السحوبات، الحلاقون، الموردون، التقارير، الإعدادات.
- شريط فلاتر عام أعلى مساحة العمل: الفرع، الفترة، من، إلى.
- لوحة التحكم تعرض مؤشرات مالية مباشرة وبطاقة أداء حلاق وجدول آخر الإدخالات.
- قسم الموردين يحتوي قائمة الموردين، صفحة حساب مورد، فواتير، سندات قبض، ونماذج إضافة.

## قرارات التصميم
- الاتجاه RTL كامل، والنصوص عربية.
- اللون الأساسي أخضر داكن `#1B4332` مع أبيض ورمادي فاتح، ولمسات ذهبية وزرقاء وحمراء للحالات المالية.
- البطاقات بزوايا 8px حتى تبقى الواجهة إدارية ومهنية.
- الجداول قابلة للتمرير أفقيًا على الشاشات الصغيرة.
- شاشة المورد تبقي الرصيد المستحق واضحًا دائمًا لأن هذا هو القرار المالي الأهم.

## الملفات
- `index.html`: النموذج التفاعلي.
- `styles.css`: الهوية البصرية والتجاوب.
- `app.js`: التنقل بين الشاشات والتبويبات.
- `assets/noorsalon-mark.png`: علامة مؤقتة للهوية.
