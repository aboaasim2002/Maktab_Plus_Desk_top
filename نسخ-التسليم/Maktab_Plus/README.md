# مكتب بلس

منصة ويب متعددة المنشآت مبنية باستخدام Next.js وPostgreSQL لإدارة مكاتب الخدمات باشتراك شهري.

## البنية

- `platform_owner`: مالك المنصة، يدير المنشآت والاشتراكات.
- `office_owner`: مالك المنشأة، وله كامل الصلاحيات داخل منشأته فقط.
- `user`: موظف تابع للمنشأة بصلاحيات يحددها مالك المنشأة.
- كل عميل وعملية وسند وفاتورة وخدمة وإعداد وسجل تدقيق مرتبط بـ `organization_id`.
- تسجيل الدخول إلى المنشأة يتوقف تلقائيًا عند تعليق أو انتهاء الاشتراك.
- يستطيع مالك المنصة الدخول إداريًا إلى أي منشأة دون معرفة كلمة مرور صاحبها، وتُسجّل العملية في سجل التدقيق.
- يستطيع مالك المنصة إعادة تعيين كلمة مرور مالك المنشأة أو أي موظف عند نسيانها.

## الإعداد

يتطلب Node.js 22 أو أحدث وPostgreSQL.

أنشئ ملف `.env.local`:

```env
DATABASE_URL=postgresql://maktab_plus:CHANGE_ME@127.0.0.1:5432/maktab_plus
DATABASE_SSL=0
COOKIE_SECURE=0
MAIN_OWNER_USERNAME=mainowner
MAIN_OWNER_PASSWORD=CHANGE_ME
BACKUP_SIGNING_SECRET=CHANGE_TO_A_RANDOM_SECRET_OF_AT_LEAST_32_CHARACTERS
PG_DUMP_PATH=/usr/bin/pg_dump
```

> لا ترفع كلمة مرور مالك المنصة إلى GitHub. تُستخدم متغيرات مالك المنصة لإنشاء الحساب عند تهيئة قاعدة بيانات جديدة فقط.

ثم:

```bash
npm install
npm run dev
```

يفتح التطبيق على `http://localhost:3000`.

## الإنتاج

```bash
npm run build
npm start
```

استخدم اتصال PostgreSQL مشفرًا واضبط `DATABASE_SSL=1` إذا كان مزود قاعدة البيانات يتطلب SSL. عند تشغيل الموقع خلف HTTPS اضبط `COOKIE_SECURE=1`.

## نقل بيانات النسخة القديمة

يستطيع مالك المنشأة رفع ملف SQLite القديم من لوحة التحكم. تُنقل بيانات العملاء والعمليات والسندات إلى منشأته وحدها، مع معالجة العملاء المكررين حسب رقم الهاتف.

## النسخ الاحتياطي والاسترداد

- مالك المنشأة فقط، أو مالك المنصة أثناء دخوله الإداري إليها، يستطيع تنزيل نسخة SQL لبيانات المنشأة.
- ملف SQL الخاص بالمنشأة موقّع من الخادم ويحتوي على بياناتها التشغيلية فقط.
- عند الاسترداد يتحقق الخادم من توقيع النسخة وأنها صادرة من المنشأة نفسها، ثم يفرض `organization_id` الحالي على جميع السجلات.
- لا يمكن استرداد نسخة منشأة داخل منشأة أخرى، مما يمنع اختلاط بيانات العملاء.
- يستطيع مالك المنصة تنزيل نسخة PostgreSQL SQL كاملة لجميع المنشآت والمستخدمين من لوحة مالك المنصة.
