# برنامج سندات الصرف والقبض

تطبيق سطح مكتب مبني باستخدام Electron و Next.js لإدارة العملاء والعمليات وسندات القبض والصرف.

## المميزات

- إدارة العملاء وبيانات التواصل.
- تسجيل العمليات المالية على العملاء أو على المكتب.
- إصدار سند قبض وسند صرف.
- تقارير وكشوف حساب.
- إعدادات بيانات المكتب.
- نسخ احتياطي واستيراد قاعدة البيانات.
- نسخة تجريبية قابلة للتوزيع بمدة محددة.

## التطوير

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build:next
npm run pack
```

لا يتم رفع ملفات البناء النهائية أو قواعد البيانات أو مجلد `node_modules` إلى GitHub.
